一、简介
RetroBoy NES 游戏合集是一套面向 ES-DE 与 RetroArch 用户整理的高质量 NES 游戏包。
合集通过程序化规则进行筛选和校验，以维基百科游戏列表作为基准，按日本、北美、
欧洲等地区清晰分类，尽量做到官方发行游戏完整覆盖、不遗漏、不混乱。
本合集不仅整理游戏文件，也包含封面、说明书等媒体信息，方便你在前端中获得更
完整、更美观的浏览体验。所有官方 ROM 均不做内容修改，只进行外部文件命名和
目录整理，确保可追溯、可核对、可持续更新。
如果你想快速搭建一个结构清晰、资料完整、持续维护的 NES 游戏合集，RetroBoy
可以帮你节省大量筛选、核对、命名和媒体整理时间。

三、安装说明
本游戏合集采用“基础包 + 平台包”的覆盖安装方式。请务必先解压 _BASE.zip，再
解压 NES.zip。
1. 请先关闭 ES-DE、RetroArch，以及其他可能正在使用相关目录的程序。
2. 解压基础包 _BASE.zip 到目标目录。如果系统提示是否合并文件夹，请选择“是”或
   “合并”。
3. 解压当前平台包 NES.zip 到同一个目标目录。如果系统提示文件或文件夹已存在，请
   选择“覆盖”。
4. 以安装到 D 盘根目录为例，基础包和平台包都应解压到 D:\，解压完成后的关键
   路径应类似以下内容。
| 项目                | 路径                                      |
| ES-DE 主目录        | D:\ES-DE                                  |
| NES 游戏目录        | D:\ES-DE\ROMs\NES                        |
| NES-JPN 游戏目录    | D:\ES-DE\ROMs\NES\NES-JPN                |
| NES-USA 游戏目录    | D:\ES-DE\ROMs\NES\NES-USA                |
| NES-EUR 游戏目录    | D:\ES-DE\ROMs\NES\NES-EUR                |
| RetroArch 目录      | D:\ES-DE\Emulators\RetroArch-Win64       |
| NES 游戏列表        | D:\ES-DE\ES-DE\gamelists\NES             |
| NES 媒体目录        | D:\ES-DE\ES-DE\downloaded_media\NES      |
5. 安装顺序非常重要。如果先解压 NES.zip 再解压 _BASE.zip，可能会导致平台配置、
   主题、游戏列表或模拟器配置不完整。
6. 解压完成后，重新启动 ES-DE。
7. 如果启动后没有看到新游戏，或游戏列表没有刷新，请在 ES-DE 中重新扫描游戏列表，
   或重启 ES-DE 后再次检查。
8. 如果你已经安装过旧版本平台包，可以直接使用新版本平台包覆盖安装，通常不需要
   删除旧文件，除非更新说明中特别要求。

八、常见问题
Q1. 我应该先解压哪个文件？
A1. 请先解压 _BASE.zip，再解压 NES.zip。NES.zip 需要覆盖安装到 _BASE.zip 解压后
    的同一个目录。

Q2. 覆盖安装时提示文件已存在，应该怎么办？
A2. 请选择覆盖。平台包中的部分配置、游戏列表和资源文件需要覆盖基础包中的默认
    内容。

Q3. 为什么有些原始文件名称、修改后文件名称和维基百科条目不完全一样？
A3. 维基百科条目用于说明游戏在参考清单中的名称。原始文件名称用于版本追溯，
    修改后文件名称用于前端显示、文件系统兼容性和统一命名规则。

Q4. 原始文件列表有什么用？
A4. 原始文件列表主要用于问题排查和版本追溯。如果你反馈某个游戏无法运行、名称
    不正确或资源不匹配，请尽量同时提供地区、修改后文件名称和原始文件名称。

Q5. 安装完成后没有看到游戏怎么办？
A5. 请确认 _BASE.zip 和 NES.zip 解压到了同一个目录，确认解压时选择了覆盖，然后
    重启 ES-DE，或在 ES-DE 中重新扫描游戏列表。
